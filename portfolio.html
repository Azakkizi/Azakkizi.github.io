Unitopia

   My friends and I designed a website for our dream university. This website is a classic university website that contains information about the university, faculties, student clubs, weekly menu, contact, and allows users to publish their own blog posts on the blog page. In the admin panel, the admin has the ability to add and edit announcements and the menu list, as well as the authority to approve or disapprove blog posts entered by others. Other users, such as students or faculty members, can add blog posts.
   Both front-end and back-end codes are written on the user interface and admin panel using HTML, CSS, JavaScript, JQuery, Node.js, and PostgreSQL. Note: It was done as a homework assignment, it is not live.

BookPlace

   My friends and I designed a book review website. People can read the summary of the books they are curious about and vote on the book and leave a review on the book if they wish by entering the website.
   The front-end and back-end codes were developed using XHTML, CSS, Java, JSF, and Apache Derby. Note: It was made as a homework assignment and is not published.

Joggernaut

   We designed this mobile application with my friends. It is not only a pedometer application for athletes but for everyone. It allows you to track your personal physical activity, set goals for yourself and achieve them, collect flags randomly placed by the application on the map around you, and aim to do more sports by competing with others through a score table created with this information.
   The front-end and back-end codes were developed using Dart, Flutter and Google Firebase. Note: It was made as a homework assignment and is not published.
